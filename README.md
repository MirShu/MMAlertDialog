# MMAlertDialog
一款仿ios的dialog弹窗
Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
Step 2. Add the dependency
```
dependencies {
	        implementation 'com.github.mamumu:MMAlertDialog:10'
	}
```
